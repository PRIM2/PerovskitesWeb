Cálculo de tolerancia

Factor de Tolerancia Extendido

La fórmula extendida para el factor de tolerancia de Goldschmidt es: \[ \alpha = \frac{r_{A,\text{eff}} + r_{X,\text{eff}}}{\sqrt{2} \left( r_B + 0.5 h_{X,\text{eff}} \right)} \] donde:

  • \(r_{A,\text{eff}}\) es el radio efectivo del cation A
  • \(r_B\) es el radio del cation B
  • \(r_{X,\text{eff}}\) es el radio efectivo del anión X
  • \(h_{X,\text{eff}}\) es el radio del anión X
Se debe tener en cuenta la globularidad del Catión A para mayor precisión.
